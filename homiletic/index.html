Întrebări frecvente

Ai întrebări? Avem răspunsuri.

De ce am nevoie de o cheie API Groq?
ResearchMind Homiletic folosește inteligența artificială Groq pentru a analiza sursele și a genera raportul de cercetare. Serverul aplicației are o limită de procesare partajată între toți utilizatorii activi. Când mai mulți preoți generează un raport în același timp, pot apărea întârzieri de 1-2 minute.

Prin introducerea propriei chei gratuite Groq, tu primești o limită dedicată, separată — raportul tău se generează imediat, indiferent de ceilalți utilizatori.
Cum obțin cheia Groq? Este cu adevărat gratuită?
Da, complet gratuit, fără card bancar. Durează aproximativ 2 minute:
  1. Mergi la console.groq.com și creează un cont (cu GitHub sau email)
  2. Apasă API Keys → Create API Key → dai un nume orice
  3. Copiezi cheia generată (începe cu gsk_) și o lipești în aplicație la secțiunea „Cheie API Groq" din sidebar
Cheia mea API este în siguranță?
Da. Cheia ta este transmisă direct și criptat de la browser-ul tău la serverele Groq — ResearchMind nu o vede, nu o stochează și nu o accesează. Cheia dispare automat la reîncărcarea paginii. Poți verifica oricând sau revoca cheia din contul Groq.
Ce se întâmplă dacă nu introduc nicio cheie?
Aplicația funcționează în continuare — va folosi cheia serverului ResearchMind, partajată între toți utilizatorii. În orele de vârf, raportul poate dura cu 1-2 minute mai mult sau poate apărea un mesaj de așteptare. Pentru utilizare optimă, recomandăm introducerea propriei chei gratuite.
Ce este Groq și de ce nu OpenAI sau alt AI?
Groq este o platformă AI care rulează modelul Llama 3.3 (Meta) extrem de rapid — de 5-10× mai repede decât OpenAI pentru texte lungi. Acest lucru înseamnă că raportul tău de cercetare se generează în câteva zeci de secunde, nu în minute. Planul gratuit Groq este mai mult decât suficient pentru uz homiletic.
